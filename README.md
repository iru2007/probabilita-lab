# Probabilita Lab

Sito web interattivo per una presentazione di matematica sulla probabilita.

Realizzato da Yapa, Piccoli, Diaco e Sanchez.

## Come usarlo

Apri `index.html` nel browser. Il sito funziona offline e contiene:

- simulatori per dado e monete;
- calcolatori per probabilita classica, eventi contrari, Venn, Bayes e Bernoulli;
- esempi guidati tratti dal lavoro sulla probabilita;
- quiz finale con punteggio.
